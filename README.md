# 🌦️ Weather App

A clean, responsive, and user-friendly **Weather Application** built using **HTML, CSS, and JavaScript**. It fetches real-time weather information using a weather API and displays current weather conditions for any city entered by the user.

---

## ✨ Features

- 🌍 Search weather by city name
- 🌡️ Displays current temperature
- ☁️ Shows weather condition and description
- 💧 Displays humidity level
- 💨 Shows wind speed
- 🌅 Weather icons based on current conditions
- 📱 Fully responsive design
- ⚡ Fast and lightweight interface
- ❌ Handles invalid city names gracefully

---

## 🛠️ Built With

- **HTML5** – Structure
- **CSS3** – Styling & Responsive Design
- **JavaScript (ES6)** – Logic & API Integration
- **OpenWeatherMap API** (or your API provider)

---

## 🔑 API Setup

1. Create a free account on **OpenWeatherMap**.
2. Generate your API key.
3. Replace the API key inside `script.js`.

```javascript
const API_KEY = "YOUR_API_KEY";
```

---

## 📋 How It Works

1. Enter the name of a city.
2. Click the **Search** button.
3. The app sends a request to the Weather API.
4. The API returns real-time weather data.
5. Weather information is displayed on the screen.
